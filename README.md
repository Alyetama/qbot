## ü§ñ Commands

- To save a quote: `.qa <quote>`
- To get a quote by keyword: `.q <keyword>`
- To get a quote info by id: `.qinfo <quote_id>`
- To delete a quote: `.qdel <quote_id>`
- To get a random quote: `.qrand`
